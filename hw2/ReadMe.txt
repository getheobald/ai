The following Python packages are required for this assignment (recommended version numbers included):

pygame==2.6.1
numpy==1.24.3
tqdm==4.65.0
gym==0.26.2


The environment used for Lab 1 (gymnasium) should also work for this assignment, with the exception of tqdm, which we did not use before.

If using an environment manager like virtualenv or conda, ensure you activate your environment first. 
You can install missing packages using pip with the following command format within your terminal or command prompt. 

$ pip install package_name==version_number

If you encounter any issues during installation, please reach out to the course staff, and we will be happy to assist.
